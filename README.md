# simple-tg-form

Простой статический сайт с формой обратной связи, отправляющей сообщения в Telegram-чат через бота.

## Быстрый старт

1. Склонируйте репозиторий и откройте его в любом статическом сервере (напр. `Live Server` в VS Code).
2. В файле `script.js` замените значения констант `BOT_TOKEN` и `CHAT_ID` на реальные:
   - `BOT_TOKEN` — токен, который выдаёт `@BotFather`.
   - `CHAT_ID` — идентификатор чата или пользователя, куда нужно отправлять сообщения. Узнать можно через `@userinfobot` или через метод `getUpdates`.
3. Обновите Telegram бота командой `/start`, чтобы он мог писать вам.
4. Откройте `index.html` в браузере и отправьте тестовое сообщение.

> ⚠️ Не храните реальные токены в публичных репозиториях или на клиентской стороне продакшн-приложений. Для боевого использования настройте ваш сервер-прокси, который будет принимать форму и отправлять запросы к Telegram API уже с сервера.
