# simple-tg-form

Простий статичний сайт з формою зворотного зв'язку, яка відправляє повідомлення в Telegram-чат через бота.

## Швидкий старт

1. Склонуйте репозиторій і відкрийте його в будь-якому статичному сервері (напр. `Live Server` в VS Code).
2. У файлі `script.js` замініть значення констант `BOT_TOKEN` і `CHAT_ID` на реальні:

   * `BOT_TOKEN` — токен, який видає `@BotFather`.
   * `CHAT_ID` — ідентифікатор чату або користувача, куди потрібно відправляти повідомлення. Дізнатися можна через `@userinfobot` або через метод `getUpdates`.
3. Оновіть Telegram-бота командою `/start`, щоб він міг писати вам.
4. Відкрийте `index.html` в браузері і надішліть тестове повідомлення.

> ⚠️ Не зберігайте реальні токени у публічних репозиторіях або на клієнтській стороні продакшн-додатків. Для бойового використання налаштуйте ваш сервер-проксі, який прийматиме форму і відправлятиме запити до Telegram API вже з сервера.
